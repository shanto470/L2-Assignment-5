Question-1: What are the different ways to select an element in the DOM?

We can use getElementById() for ID selection, getElementsByClassName() and getElementsByTagName() for class and tag selection, and querySelector() or querySelectorAll() for CSS-style selection

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML retrieves or sets the HTML inside an element, innerText retrieves only visible text, and textContent gets all text, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM?

It allows handling events efficiently by attaching a listener to a parent element and detecting which child triggered the event using event.target, reducing the need for multiple event listeners.

Question-4: What is event bubbling in the DOM?

When an event occurs on a nested element, it first triggers on the target element and then propagates upwards to its ancestors unless event.stopPropagation() is used.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement(), add them with appendChild(), and remove them using remove() or removeChild(), modifying the DOM dynamically.
